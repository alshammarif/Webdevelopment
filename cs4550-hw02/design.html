Design Document:

User-First Design:

  • As a new User, I should be able to register a new account:
    • User visits the home/main page
    • User clicks "sign up" button/link
    • User enters their email, name, and password in the correct fields
    • User then submits the form and awaits a confirmation email
    • User confirms their email and is free to log into their created account and use the website

  • As a user, I should be able to update my information
    • User clicks the link "Profile" that takes them to the user profile page
    • User then checks the information before clicking "edit"
    • User changes any or all fields
    • User saves all changes or cancels request
    • User is take back to the Profile page to see updated information

  • As a user, I can create a new post
    • User logs into account and is taken to a personal home page/dashboard
    • User clicks "New Post" button
    • User is taken to a new post page
    • User enters a message in the text box
    • User then clicks "Post" to save the message and post it. Or "cancel" to cancel request to post
    • User is taken back to personalized homepage to see their post has been added to the queue of posts

  • As a user, I should be able to follow another user
    • User clicks the search bar or "maginifying glass"
    • User enters the username of the other user
    • User selects the user in the list of users
    • User is taken to user homepage
    • User clicks "follow" and is now changed to "unfollow"
    • User can now see followed users posts on their homepage/dashboard

  • As a user, I should be able to edit/delete an old post
    • User finds wanted post
    • User clicks small menu/arrow selects "edit" or "delete"
    • User deletes post gets a warning before following through with the deletion
    • User editing post is taken to an editing page
    • User edits the message before clicking "save" or "cancel" to commit changes to the post

  • As a user, I should be able to tag another user in a post
    • User selects post and clicks "edit" or creates a new post
    • User then finds the part of the post to tag the other user
    • To tag another user, the User must put '@username' in the text box
    • User can save changes to old post or post the new post
    • The other user gets a notification of the tag
    • User has now tagged another user

  • As a user, I can add hashtags to the post
    • User selects post and clicks "edit" or creates a new post
    • User then finds the section of the post to add a hashtag
    • To create a hashtag, User must start with '#' and message i.e #awesome
    • User saves all changes or creates a new post
    • User has now added wanted #hashtags to wanted post
